# Development Guide

## Prerequisites

- [Node.js](https://nodejs.org/) via nvm

## Quick Start

```bash
nvm use
npm install
npm start
```

The app will be available at http://localhost:3000

## Additional Scripts

- `npm run build` - Creates a production build
- `npm run deploy` - Deploys to Permaweb (requires `DEPLOY_ANT_PROCESS` env var)
